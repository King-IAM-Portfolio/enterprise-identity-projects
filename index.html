About This Portfolio

This portfolio is designed to demonstrate practical Identity and Access Management skills using Microsoft Entra ID. Each project is structured around a realistic enterprise identity scenario and includes business requirements, implementation steps, validation notes, security considerations, screenshots, and lessons learned.

The goal is to show hands-on experience with identity administration, access control, governance, application security, monitoring, and Zero Trust implementation in a lab environment.
